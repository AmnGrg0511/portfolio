# Portfolio

Personal portfolio site built with React and Material-UI. The project focuses on a crisp, sectioned layout with animated reveals, smooth navigation, and a deploy flow to GitHub Pages.

## Highlights
- Multi-section layout for hero, work, skills, and contact
- Smooth scrolling navigation via `react-scroll`
- Motion-driven transitions powered by `framer-motion`
- GitHub Pages deployment with `gh-pages`

## Tech Stack
- React 17 + Create React App
- Material-UI v4
- Framer Motion
- React Scroll

## Scripts
- `npm start` - Run the dev server
- `npm run build` - Production build
- `npm test` - Test runner
- `npm run deploy` - Publish to GitHub Pages

## Deployment Notes
- `homepage` in `package.json` controls the public URL for the GitHub Pages build
- Deployment runs `npm run build` before publishing the `build` folder
